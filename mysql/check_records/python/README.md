## Setup

Setup venv enviroment.

```sh
python -m venv .venv
source ~/.venv/bin/activate
pip install -U pip
pip install -r requirements.txt
```

Copy .env.sample to .env.

```sh
cp .env.sample .env
```

## Usage

```sh
python main.py 
```
